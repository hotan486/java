## 연산자와 연산식

- 연산(operations) : 데이터를 처리하여 결과를 산출하는것 
- 연산자(operator) : 연산에 사용되는 표시나 기호 ex) /,*,+,-
- 피연산자(operand) : 연산되는 데이터 ex) x,y,z
- 연산식(expressions) : 연산자와 피연산자를 이용하여 연산의 과정을 기술한 것 ex) x + y  

        x = 2;
        y = 3;
        z = x + y;
