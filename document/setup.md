# 개발환경

## 깃 크라켓 사용법 

- https://www.gitkraken.com/
- 깃 허브 저장소에서 소스를 가져오는 경우 

    1. 깃 크라켓에서 file 아이콘 클릭 
    2. clone후 가져올 저장소 선택 
    3. 로컬에서 저장할 위치 선택
    4. 생성하면 로컬 저장소 만들어짐    

- 로컬에서 생성된 소스 깃에 올리기 

    1. 먼저 로컬에서 생성된 프로젝트를 깃 저장소로 만들기 위해 
    2. 로컬에서 cmd에서 git init
    3. 깃 크라켄에서 open
    4. 스테이지 커밋
    5. 푸시하면 리모트 입력
